### 확정 사항

**인트로 페이지 (`index.html`)**
- 직업 선택: 풀스크린, 가로 카드 5개 세로 나열 (이미지 마스크 + 직업명), 스크롤 없음
- 무기 선택: 동일 카드 스타일, 세로 스크롤 가능, 상단에 `← 직업명` 뒤로가기
- 단계 전환: 좌우 슬라이드 애니메이션

**URL 구조**
- 인트로: `/`
- 공략: `/guide.html?class=전사&weapon=sword` (메인 빌드 자동 로드)
- 빌드 전환 시 URL pushState 업데이트: `?class=X&weapon=Y&build=sword_전사_1`

**공략 페이지 (`guide.html`)**
- 상단 고정 2줄: `[무기·직업] [빌드 드롭다운▾]` / `[탭 드롭다운▾]`
- 좌우 스와이프: 빌드 전환
- 상하 스와이프: 탭 전환 (콘텐츠가 1페이지에 맞게 설계하므로 스크롤 충돌 없음)
- 드롭다운 + 스와이프 동시 제공

**스키마 변경**
- 빌드 JSON에 `isMain: true` 필드 추가 (없으면 `_0` 빌드가 기본)
- `isMain: true`인 빌드를 `guide.html` 진입 시 자동 로드
